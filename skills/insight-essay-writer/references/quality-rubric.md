# Quality Rubric v4

Use this for final drafts and review mode.

| Dimension | 10/10 | 7/10 | 4/10 |
|---|---|---|---|
| Thesis sharpness | Specific, surprising, decision-changing | Clear but familiar | Broad topic statement |
| Source transformation | Material becomes a new argument | Mostly organized well | Reads like recap |
| Source-relative originality | Clear new angle against reference | Some overlap but new parts exist | Looks like smoother rewrite |
| Reader self-relevance | Opening makes reader feel the stakes | Relevance emerges later | Reader payoff unclear |
| Reader actionability | Reader can change a concrete decision | Advice present but broad | Ending inspirational only |
| Reader-gain artifact | Useful table/checklist/model tied to thesis | Artifact exists but generic | No usable artifact |
| Narrative smoothness | Scene → mechanism feels natural | Some framework drag | Notes stitched together |
| Concept restraint | One spine, few concepts | Several concepts, readable | Taxonomy overload |
| Title quality | Natural, specific, curiosity-creating | Clear but stiff | Internal framework label |
| Evidence and boundary | Strong details and edge cases | Enough support | Thin evidence or no limits |
| Chinese naturalness | Direct, concrete, publishable | Mostly smooth | Generic AI cadence |

Target: 90+. Minimum publishable score: 88.

## Hard gates

- Source-relative originality < 8: change opening, structure, thesis mode, or example mix.
- Reader self-relevance < 8: rewrite opening.
- Reader-gain artifact < 8: add or improve decision tool.
- Narrative smoothness < 8: reduce framework density.
- Concept restraint < 8: cut secondary mechanisms.
- Title quality < 8: regenerate titles.
- Publication hygiene failure: remove research scaffolding.

## Rewrite moves

| Weak score | Rewrite move |
|---|---|
| Thesis | Rewrite title, opening, thesis paragraph |
| Originality | Switch to adjacent or missing-angle thesis; replace first example |
| Reader gain | Add reader pain, decision change, and artifact |
| Artifact | Turn advice into a table, checklist, or operating model |
| Smoothness | Convert framework sections into causal narrative |
| Concept restraint | Pick one spine, save other ideas for future articles |
| Title | Generate 20 titles across four families |
| Evidence | Return to source map and add underused details |
| Boundary | Add where the mechanism breaks |
| Language | Shorten paragraphs, use actors and verbs |
