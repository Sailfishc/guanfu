# Source Voice Integrity Gate

Keep source relationships honest. A polished essay loses trust when it invents direct access.

## Trigger

Run before final output whenever the article uses interviews, podcasts, transcripts, reports, clippings, or someone else's draft.

## Hard rule

Only use first-person access when the author actually conducted the interview, experiment, or meeting.

## Scan phrases

```text
当我问他/她
我采访了
他/她告诉我
我们聊到
在我的访谈里
我和他/她讨论
```

## Safe replacements

| Risky phrase | Safe phrase |
|---|---|
| 当我问 Cat Wu | Lenny 在访谈里问 Cat Wu |
| Cat Wu 告诉我 | Cat Wu 在访谈里提到 |
| 我采访她时 | 这段访谈里 |
| 我们聊到 Anthropic | 材料里谈到 Anthropic |

## Final check

Every named fact should answer: how does the article know this?

Acceptable answers: source transcript, podcast, report, user-provided notes, public release, author's own interview, author's own experiment.
